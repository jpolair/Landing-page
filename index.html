Développement Front-End

Pour le développement Front-End j'utilise les langages stables les plus modernes et les plus utilisés à ce jour c'est-à-dire le HTML5, le CSS3 et bien sur le JavaScript. Pour leur richesse et leur flexibilité je peux utiliser les frameworks CSS Bootstrap et Materialize, je sais égalemnt me servir de la puissance du CSS3 pour des interfaces plus personnalisées. Pour une expérience utilisateur incomparable j'utilise le framework AngularJS (après la version 1.5 pour me servir des component ) ou Angular pour me service du typage du Typescript pour rendre les applications plus robustes. Je peux également développer pour le mobile, pour cela j'utilise Ionic.

Développement Back-End

En ce qui concerne le développement back, j'utilise là encore des technologies javascript avec la puissance du moteur V8 de google chrome côté back : NodeJs, je trouve que cette techno, combinée à une base de données noSQL comme mongoDb est très agréable à travailler. Ces technologies sont très performantes, mais je m'interesse également aux outils de type cloud ( Firebase , Azure , AWS ) je pense que les grandes compagnies qui sont derrières ces services cloud ont les moyens de nous concevoir des outils hyper puissants, et avec le temps leur concurrence nous rendra service...

Outils

Nous sommes en 2017, pour travailler dans de bonnes conditions il faut les bons outils!
