Développement Front-End

Pour le développement Front-End j'utilise les langages stables les plus modernes et les plus utilisés à ce jour c'est-à-dire le HTML5, le CSS3 et bien sur le JavaScript.Pour leur richesse et leur flexibilité j'utilise les frameworks CSS Bootstrap et Materialize, pour une expérience utilisateur incomparable j'utilise le framework MVVM AngularJs ( il est formidable ).L'aspect modularité étant arrivé après la version 1.5 a redonné une nouvelle à ce framework qui, à mon avis, sera de plus en présent ces prochaines années

Développement Back-End

Pour le développement Back-End j'utilise également des technologies très modernes, je crois en la puissance du JavaScript côté serveur : j'utilise donc NodeJS.La puissance du moteur V8 de Google mise sur un serveur permet de développer des applications très rapides et modifie considérablement l'expérience utilisateur.En ce qui concerne le système de gestion de base de données pour les gros volumes et une gestion souple je crois en la puissance du NoSQL grâce à MongoDB, toutefois, quand le NoSQL n'est pas adapté au projet ou des plus petits projets rien de tel qu'un SGBDR tel que MySQL qui n'a plus rien à prouvé
